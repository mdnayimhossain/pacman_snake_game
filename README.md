# Pac-Man Snake Game (Python)

## Overview
This is a modern **Pac-Man Snake Game** built with **Python**, using **Pygame** for the game mechanics and **Tkinter** for login/signup functionality. Each player can **create an account**, **save high scores**, and enjoy a professional gameplay experience.

This project demonstrates:

- GUI programming (Tkinter + Pygame)
- Persistent data storage with JSON
- Object-oriented and modular game design
- Animated Pac-Man style visuals
- Snake mechanics with self-collision and scoring

---

## Features

- **Login & Signup:** Players can create accounts and log in.  
- **Persistent High Scores:** Scores are saved per player in a JSON database (`data.json`).  
- **Pac-Man Snake Mechanics:** Collect food, grow in length, and avoid collisions.  
- **Animated Pac-Man Head:** Mouth opens and closes as the snake moves.  
- **Sound Effects (Optional):** Eating and game over sounds (place `.wav` files in `assets/`).  
- **Restart & Quit:** Press `R` to restart, `Q` to quit.
